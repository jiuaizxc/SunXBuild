# SunxBuild
这是一款运行在CocosCreator的自动化出包插件。


插件可以同时出包IOS、安卓、H5小游戏平台的游戏。 可以根据不同的渠道，不同平台，选择自己需要的广告SDK、支付SDK、分析SDK、以及一些其他的SDK。 
还可以根据不同渠道，使用不同的签名、启动图、Icon图标。
